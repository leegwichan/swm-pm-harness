## 한 줄 소개

> "AI 기반 개인 맞춤 의상 추천 플랫폼. 매일 아침 사용자의 옷장, 날씨, 일정, 기분을 분석해 최적의 옷을 추천해준다."

## 타겟 유저

- 20대 후반~30대 초반 직장인
- 매일 아침 옷 고르는 데 10~20분 쓰며 스트레스 받는 사람
- 옷은 많은데 항상 같은 것만 입는 사람

## 핵심 기능 (MVP 범위)

1. **옷장 등록**: 사진 업로드 → AI가 자동으로 카테고리·색상·스타일 분류
2. **데일리 추천**: 매일 아침 알림 → 3가지 코디 제안
3. **기분 연동**: 사용자가 오늘의 기분(5지선다)을 선택하면 그에 맞는 톤으로 추천
4. **날씨 연동**: 기상청 API
5. **일정 연동**: Google Calendar 연동 → 중요 미팅이면 포멀하게
6. **피드백 학습**: 사용자가 "좋아요/별로" 피드백 → 모델 재학습
7. **SNS 공유**: 오늘의 코디를 인스타그램 스토리로 공유
8. **커뮤니티**: 다른 사용자 코디 구경하기
9. **쇼핑 연동**: 부족한 아이템을 쇼핑몰에서 바로 구매

## 기술 스택 논의

- **프론트**: React Native (iOS/Android 동시 출시)
- **백엔드**: FastAPI + PostgreSQL + Redis
- **인프라**: AWS EKS (Kubernetes), ArgoCD로 GitOps
- **ML**:
  - 옷장 분류: CLIP 파인튜닝
  - 추천: Two-tower 추천 모델 자체 개발
  - 개인화: 사용자별 LoRA 어댑터 유지
- **기타**: 실시간 이미지 분석을 위한 gRPC 내부 통신, Kafka 이벤트 스트리밍
